# dashboard_gamepass_sales
Xbox Game Pass – Subscription Sales Dashboard

Este repositório contém um dashboard em Excel desenvolvido para análise de vendas de assinaturas do Xbox Game Pass. O arquivo consolida dados de clientes, planos, receitas adicionais e métricas de desempenho em uma visualização executiva.

Estrutura do Arquivo

O arquivo Excel está organizado nas seguintes abas:

Assets

Aba destinada aos recursos visuais do dashboard, como paleta de cores e códigos HEX, garantindo padronização visual.
Não contém dados analíticos.

Bases

Base de dados principal do projeto.

Principais campos:

Subscriber ID

Subscriber Name

Plan (Core, Standard, Ultimate)

Subscription Start Date

Auto Renewal

Subscription Type (Monthly, Quarterly, Annual)

Subscription Price

Add-ons (EA Play Season Pass, Minecraft Season Pass)

Coupons

Total Value

Esta aba é a fonte de dados para todas as análises.

Cálculos

Aba responsável pelos cálculos e consolidações.

Tabelas dinâmicas

Agregações de receita

Análises por plano, tipo de assinatura e renovação automática

Funciona como camada intermediária entre a base e o dashboard.

Dashboard

Aba de visualização final.

Título: Xbox Game Pass Subscription Sales

Período analisado: 01/01/2024 a 31/12/2024

Indicadores e gráficos de desempenho

Layout focado em leitura rápida e tomada de decisão

Objetivo do Projeto

Analisar o desempenho das vendas de assinaturas

Avaliar o impacto de add-ons e cupons

Facilitar a visualização de métricas de receita

Servir como modelo de dashboard comercial em Excel

Como Utilizar

Abra o arquivo no Microsoft Excel

Atualize os dados na aba Bases

Verifique a atualização dos cálculos

Analise os resultados no Dashboard

Observações

Alterações na estrutura da aba Bases podem exigir ajustes nos cálculos ou gráficos

Projeto voltado para estudo, portfólio ou adaptação para cenários reais
